# Knowledge AI Platform

B2B Educational Intelligence Platform - Multi-source content monitoring & analysis

## Overview

A comprehensive platform for automated sector monitoring and content analysis, designed for educational intelligence and knowledge management.

## Tech Stack

- **Frontend**: React 19 + Vite + TypeScript + Tailwind CSS
- **Backend**: Node.js + Hono
- **Database**: Supabase (PostgreSQL + Auth + Storage)
- **AI**: Google Gemini 2.5 Pro/Flash
- **Automation**: N8N Cloud

## Getting Started

```bash
npm install          # Install dependencies
npm run dev          # Start frontend dev server
npm run server       # Start backend server
npm run dev:all      # Start both frontend and backend
```
